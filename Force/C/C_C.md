# C语言

## 1.C语言的历史

### 1.1.C语言与人

C语言之父：肯.汤普森，丹尼斯.里奇

**Fortran（人类历史上第一个高级程序语言）-->BCPL-->B语言-->初代的C**

C语言的参考书：K&R（C语言黑皮书）

BCPL和B都支持指针间接方式，C支持了指针（指针是C语言的灵魂，是使C语言成为C语言的基石）

C语言收到PL/1的影响，也致力于表达操作系统所使用的机器指令。

1973年3月，UNIX的第三版上出现了C语言的编译器

1973年11月，第四版UNIX操作系统完全由C语言重写。
